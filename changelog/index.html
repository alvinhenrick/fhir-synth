    CHANGELOG¶

    v1.5.0 (2026-03-11)¶

    Features¶

    • Add comprehensive guidelines for patient variation, comorbidity patterns, and realism in FHIR resource generation (2653c2d)

    • Add fix_naive_date_times function to handle naive datetime patterns (db31ab1)

    • Enforce timezone-aware DateTime handling and update related documentation (b3fb3c1)

    • Enhance architecture diagram with improved formatting and connections (3350977)

    • Enhance patient resource generation with realistic demographics and encounter details (3ff5f2c)

    • Implement EMPI prompt generation and FHIR resource validation (c690d1d)

    • Improve code formatting and readability in cli.py and fhir_spec.py (bf7a352)

    • Refine FHIR spec documentation and error handling guidelines (bf484dc)

    • Remove build_bundle_code_prompt function and update related references (cb3a905)

    • Remove build_rules_prompt function and related references (584a95d)

    • Remove fix_naive_date_times function and update related references (3b0baf0)

    • Remove print_quality_report function and simplify grade calculation logic (f3a2bd8)

    • Remove unused imports for improved code cleanliness in constants.py, fhir_spec.py, manager.py, and test_prompts.py (e87a7ac)

    • Support multiple FHIR versions (R4B, STU3) in prompts and documentation (fa0005e)

    • Update architecture diagram with enhanced theme variables for improved clarity (5022f7d)

    • Update hard rules and EMPI linkage requirements for clarity and precision (da0a556)

    • Update validation step to use model_dump with exclude_none and json mode (4a0326e)
